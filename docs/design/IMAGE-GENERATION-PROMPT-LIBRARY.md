# PSL One — Image Generation Prompt Library

Safe, non-infringing prompts for AI image generation tools (Midjourney, DALL-E, Stable Diffusion, Firefly).

All prompts are designed to avoid player likenesses, official logos, and copyrighted content.

---

## Usage Rules

1. Never generate images of identifiable real people (players, managers, celebrities)
2. Never generate official logos or crests
3. Never generate imagery that implies gambling or real-money activity
4. Always review output before publishing
5. Label all AI-generated assets as AI-generated in internal asset registry

## Tips for Better Results

1. Add `--ar 16:9` (Midjourney) or specify dimensions for correct aspect ratios
2. Add `--no logos, no text` to prevent unwanted text in output
3. Add `--style raw` in Midjourney for more photorealistic results
4. For card backgrounds: add `--ar 5:7` for portrait card format
5. Always request multiple variations and choose the safest/most suitable
6. Test with white text overlay before accepting an asset
