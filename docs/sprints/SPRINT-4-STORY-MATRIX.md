## Risk Register

| Risk | Stories Affected | Mitigation |
|------|-----------------|------------|
| External provider API keys not available in time | S4-06 | Spike uses mocked HTTP responses; real key wired in tools/ only, never committed |
| Backend contract gaps larger than S4-03 audit reveals | S4-04, S4-05, S4-07 | Agent 4 completes full 49-route audit before Agents 5/6/8 start; 501 stubs are acceptable deliverables |
| Vercel project not yet created under owner account | S4-01 | Stop at credential step per gate — Agent 2 documents steps, owner executes |
| Auth rework touches `getBetaToken()` pattern | S4-07 | Agent 8 must not replace `getBetaToken()`; extend only |
| New backend endpoints accidentally activate PSL | S4-04, S4-07 | All new routes checked against season gate before merge; `assertSeasonActive` guard applied |
| Analytics payload inadvertently captures PII | S4-08 | Privacy classification column mandatory in event catalogue; PII = instant block |
| Test count regression (currently 1,560 passing) | S4-09 | Release gate fails if total test count drops below 1,560 |
